# TimeFlow

- Track progress between two dates 
- Android Widget to visualize progression

# Install

1. Download APK from [latest release](https://github.com/tobiaswuerth/TimeFlow/releases/tag/v1.0)
2. Install APK

Requires Android >=15.0.
